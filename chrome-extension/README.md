# DevSync Chrome 扩展

在任意本地前端页面上拖拽调整元素位置和大小，并同步到本地文件。

**无需修改任何项目代码！**

## 安装插件

1. 打开 Chrome 浏览器，访问 `chrome://extensions/`
2. 开启右上角的「**开发者模式**」
3. 点击「**加载已解压的扩展程序**」
4. 选择 `chrome-extension` 文件夹
5. 点击扩展详情，开启「**允许访问文件网址**」

## 使用方法

### 方式一：仅编辑（不同步到文件）

1. 直接用 Chrome 打开任意本地 HTML 文件
2. 点击 DevSync 插件图标 → 「启用编辑模式」
3. 拖拽/缩放元素（刷新后会重置）

### 方式二：编辑并同步到本地文件

1. **启动同步服务器**（只需运行一次，可处理任意项目）
   ```bash
   node sync-server.js
   ```

2. 用 Chrome 打开任意本地 HTML 文件
3. 点击 DevSync 插件图标 → 「启用编辑模式」
4. 拖拽/缩放元素 → **自动保存到文件**

## 对前端开发者的要求

**无！** 

- 不需要在项目里添加任何代码
- 不需要引入任何脚本
- 不需要安装任何依赖
- 只需要一个通用的 `sync-server.js` 在后台运行

## 文件说明

```
chrome-extension/
├── manifest.json      # 插件配置
├── popup.html/js      # 插件弹窗
├── content.js/css     # 注入页面的代码
├── sync-server.js     # 通用同步服务器 ⭐
└── icons/             # 图标
```

## 工作原理

```
任意 HTML 页面  ←──  Chrome 插件  ──WebSocket──▶  sync-server.js  ──▶  修改本地文件
     ↑                                                                      │
     └──────────────────────────────────────────────────────────────────────┘
```
